Introducción

Code Hammer Judge es un sistema distribuído en tiempo real diseñado para administrar y controlar maratones de programación. Diseñado como una herramienta de estudio para la Universidad Pontificia Bolivariana seccional bucaramanga.

En este manual asume que el lector es familiar con el proceso de una competencia de programación. Además, está orientado para tener una idea general de todas las funcionalidades del software, así como la respuesta a preguntas más comunes que pueden presentarse durante su uso.
